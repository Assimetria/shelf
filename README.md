# Shelf

Digital content library and knowledge management system. Capture, organise, and share resources across your team.

**Your knowledge, beautifully organised.**

## Stack
React 18 + Webpack 5 + Node.js/Express + PostgreSQL + SCSS + Docker

## Getting Started
```bash
npm install
npm run dev
```

## Structure
- `client/` — React frontend
- `server/` — Express backend
- `@system/` — Shared template code (managed by Frederico)
- `@custom/` — Product-specific customizations
